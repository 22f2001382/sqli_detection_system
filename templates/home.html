Data Shield : A Comprehensive Solution for SQL Injection Attacks

DataShield is designed to provide a robust and privacy-preserving solution for detecting SQL injection attacks through a federated learning approach using a Support Vector Machine (SVM) model deployed in the cloud. According to OWASP's list of top 10 vulnerabilities, SQL injection remains a significant threat to web applications, enabling attackers to manipulate databases and extract sensitive information through malicious queries. Traditional detection methods rely on centralized data collection, which raises concerns about privacy, scalability, and generalization. To address these challenges, DataShield utilizes federated learning to train the SVM model across multiple edge devices from different cloud regions, eliminating the need to transfer sensitive data to a central server. A dynamic dashboard offers real-time insights into model performance and client participation, enabling stakeholders to respond swiftly to potential threats. Its decentralized architecture not only strengthens security but also ensures adaptability to diverse environments, making it a versatile solution for modern cybersecurity needs.
